# matrix

``` r
mat1 <- matrix(1:6,2,3,byrow=T)
mat1
```

    ##      [,1] [,2] [,3]
    ## [1,]    1    2    3
    ## [2,]    4    5    6

``` r
mat4 <- matrix(1:9,3,byrow=T)
mat4
```

    ##      [,1] [,2] [,3]
    ## [1,]    1    2    3
    ## [2,]    4    5    6
    ## [3,]    7    8    9

``` r
rbind(mat1,mat4)
```

    ##      [,1] [,2] [,3]
    ## [1,]    1    2    3
    ## [2,]    4    5    6
    ## [3,]    1    2    3
    ## [4,]    4    5    6
    ## [5,]    7    8    9

``` r
rbind(mat4,11:13)
```

    ##      [,1] [,2] [,3]
    ## [1,]    1    2    3
    ## [2,]    4    5    6
    ## [3,]    7    8    9
    ## [4,]   11   12   13

``` r
cbind(mat4,11:13)
```

    ##      [,1] [,2] [,3] [,4]
    ## [1,]    1    2    3   11
    ## [2,]    4    5    6   12
    ## [3,]    7    8    9   13

``` r
seasons <- matrix(c("봄","'여름","가을","겨울"),2,2,byrow=T)
seasons[,2]
```

    ## [1] "'여름" "겨울"

행렬은 디폴트로 위에서 아래로 읽는다. byrow를 통해 설정이 가능하다.
rbind()는 행 기준으로 묶어주고 cbind()는 열 기준으로 묶어준다. 이때
자료형이 같아야 하고 길이도 같아야 에러 및 경고메시지가 뜨지 않는다.

# dataframe

``` r
no <- c(1,2,3,4)
name <- c('Apple','Peach','Banana','Grape')
price <- c(500,200,100,50)
qty <- c(5,2,4,7)
sales <- data.frame(NO=no,NAME=name,PRICE=price,QTY=qty)
sales
```

    ##   NO   NAME PRICE QTY
    ## 1  1  Apple   500   5
    ## 2  2  Peach   200   2
    ## 3  3 Banana   100   4
    ## 4  4  Grape    50   7

``` r
sales[1:3,]
```

    ##   NO   NAME PRICE QTY
    ## 1  1  Apple   500   5
    ## 2  2  Peach   200   2
    ## 3  3 Banana   100   4

``` r
sales[1:2,2:3]
```

    ##    NAME PRICE
    ## 1 Apple   500
    ## 2 Peach   200

``` r
subset(sales,price==200)
```

    ##   NO  NAME PRICE QTY
    ## 2  2 Peach   200   2

``` r
subset(sales,qty<5)
```

    ##   NO   NAME PRICE QTY

``` r
subset(sales,name=='Banana')
```
